# Hero Background Image

## Required File
Place your hero background image here as: **hero.jpg**

## Specifications
- **Filename:** hero.jpg (or hero.png, hero.webp)
- **Recommended Size:** 1920x1080 or larger
- **Format:** JPG, PNG, or WebP
- **Aspect Ratio:** 16:9 (landscape)
- **File Size:** Keep under 500KB for optimal loading

## Usage
This image is used as the background for:
- Landing page (/)
- Claim voucher page (/claim)

## Tips
- Use a high-quality image that represents your brand
- Ensure good contrast for text readability
- Consider using a gradient overlay (already applied in CSS)
- Optimize the image before uploading using tools like:
  - TinyPNG (https://tinypng.com)
  - Squoosh (https://squoosh.app)
